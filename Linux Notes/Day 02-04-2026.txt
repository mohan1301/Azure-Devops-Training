Details that needs to be hardcoded in WebVM in order to connect to DBVM
cd /var/www/html
vi db.php

Paste the below text
<?php
$host = "10.0.1.6";
$user = "blinkituser";
$pass = "StrongPassword123";
$db   = "blinkit";
 
$conn = new mysqli($host, $user, $pass, $db);
 
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}
?>
