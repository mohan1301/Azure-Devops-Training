Baremetal OS is a layer which is installed for Virtualization of Physical server to multiple Virtual Servers

Physical Server is split into Multiple Virtual Machines.
Virtual Machines are split into Multiple Containers
Each container will have packages where each package should have:
1. Application code
2. Runtime Env
3. Libraries
4. Dependencies
5. Config files
So that, the same package can be imported to another Machine and run the same application
A platform that manages all the Containers is called Docker. Alternatives are PodMan, Cri-O, Containerd, Docker
