	name: 'UCRG'

CreateRG - Logical name(your reference name)

Till now, we have written the code for single resource. now, lets write altogether at once but in 2 files(RG Alone + ASP, Webapp).
Note: We cannot combine two files where one is having Subscription as targetScope and other is having resource group as targetScope

main.bicep -  Where the application is going to run
resource.bicep - Where all the resources are going to create

targetScope = 'subscription'
resource CreateRG 'Microsoft.Resources/resourceGroups@2025-04-01' = {
	name = 'UCRG001'
	location = 'canadacentral'
}

module ucsaloonsite 'resource.bicep' = {
	name: 'deploy-webapp'
	scope: CreateRG
}

// ucsaloonsite is the logical name of the module(IAC Specific)
// CreateRG is the scope that module will look after which is defined above
// deploy-webapp is the name of the Azure Deployment name(Azure Specific)
